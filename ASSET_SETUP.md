# Configuración local de recursos visuales

La copia pública de Proyecto Origen no incluye los archivos fuente de Tiny Swords ni
otros recursos visuales de terceros cuya redistribución está restringida.

## Para restaurar el prototipo localmente

1. Obtén Tiny Swords directamente desde la página oficial de Pixel Frog.
2. Revisa y acepta la licencia vigente del creador.
3. Coloca los archivos únicamente en tu copia local y privada.
4. Conserva las rutas esperadas por las escenas del prototipo.
5. No agregues esos archivos a Git, Git LFS, releases ni archivos comprimidos.

Carpetas locales sugeridas:

```text
Assets-Private/
ThirdParty-Private/
```

Las carpetas anteriores están excluidas mediante `.gitignore`.

## Importante

Este repositorio prioriza la revisión de código, arquitectura, documentación, capturas
y video. Sin los recursos locales autorizados, algunas escenas pueden mostrar errores
de dependencias o no ejecutarse visualmente de forma completa.
